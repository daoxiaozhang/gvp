## GVP

---



> Glodon visual platform plug-in development environment



### Install

```shell
npm i -g gvp
```

### Usage

Create project

```shell
 gvp create hello # create a project where name is hello
```

Go to the director `hello`ï¼Œinstall 

```shell
cd hello 
npm install
```

Start

```shell
npm run dev 
#or
npm run dev:chart # start a chart debug mode
```

Build

```shell
npm run build
```
